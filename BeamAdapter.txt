Explications du code:
Chaque class est code par trois fichiers portant sont nom .h, .inl et .cpp.

1.BaseRestShape : decrit les fonctions de formes dans le cas d'une seule ligne droite. (en cours de developpement).

2.WireRestShape : decrit le fonctions de formes dans le cas de plusieurs segment en utilisant les abcisses curvilignes.

3.BeamInterpolation : Interpolation dans le cas non utilise les abcisses curvilignes.

3.WireBeamInterpolation : Interpolation utilisant les abcisses curvilignes (les splines), cette class herite de BeamInterpolation et pointe vers WireRestShape class pour avoir des informations des fonctions de formes.

4.AdaptiveBeamForceFieldAndMass : Pointe vers l'interface BeamInterpolation pour effectuer des interpolations que ce soit ou non dans le cas des abcisses curvilignes.

5.AdaptiveBeamController : Pointe vers l'interface BeamInterpolation pour effectuer des interpolations que ce soit ou non dans le cas des abcisses curvilignes.

6.AdaptiveBeamMapping : Pointe vers l'interface BeamInterpolation pour effectuer des interpolations que ce soit ou non dans le cas des abcisses curvilignes.

7.InterventionalRadiologyController : Pointe vers plusieurs WireBeamInterpolation pour effectuer les interpolations de splines. Chacun des WireBeamInterpolation est un instrument qui est gerer pas le controller. Ce controller permet d'interagir avec l'utilisateur.

8.MultiAdaptiveBeamMapping : Pointe vers plusieurs WireBeamInterpolation pour effectuer les interpolations de splines et vers un InterventionalRadiologyController pour adapter des changements a chaque fois le controller effectue des operations. Ce mapping prend la list des instruments du controller comme parametre et creer une liste sous-mapping de AdaptiveBeamMapping pour chacun des instruments.
