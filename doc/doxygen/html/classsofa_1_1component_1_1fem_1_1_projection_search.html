ProjectionSearch< DataTypes > Class Template Reference

Detailed Description

template<class DataTypes>
class sofa::component::fem::ProjectionSearch< DataTypes >

When the Newton method doesn't converge when searching for the closest projection on the curve, we will start a dichotomic search, for now a unoptimized brute force approach.
