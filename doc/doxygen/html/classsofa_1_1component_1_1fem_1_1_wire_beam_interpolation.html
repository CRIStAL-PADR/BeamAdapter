Detailed Description

template<class DataTypes>
class sofa::component::fem::WireBeamInterpolation< DataTypes >

AdaptiveBeam Interpolation provides the basis of the Beam computation As the computation is adaptive, the interpolation can be modified at each time step.

Compute Finite Element elastic force and mass based on Adaptive 6D beam elements.

  • Adaptive beam interpolation
  • Adaptive Force and Mass computation
  • Adaptive Mapping
